## Packages
framer-motion | For smooth page transitions and micro-interactions
lucide-react | For beautiful icons
clsx | For conditional class merging
tailwind-merge | For conditional class merging

## Notes
- Theme uses a vibrant orange/red primary color similar to Swiggy/Zomato.
- Authentication uses the `/api/user`, `/api/login`, `/api/register` endpoints.
- Products are fetched from `/api/products`.
- Orders are managed via `/api/orders`.
- Cart state is managed locally in a React Context or similar, then synced to backend on checkout.
